Event Target Test

Description

This page connects to several different different Server Sent Event (SSE) stream. Contents from each stream are sent to a single DOM element.

Example HTML

<div hx-ext="sse" hx-sse="connect:/users.html swap:message" hx-target="#otherNode"></div>
<div id="otherNode"></div>
